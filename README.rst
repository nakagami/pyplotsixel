==================
pyplotsixel
==================

Matplotlib sixel backend.

It can draw matplotlib figures in SIXEL available terminal.


Example
-----------------

::

   import matplotlib
   import matplotlib.pyplot as plt
   import numpy as np

   matplotlib.use('module://pyplotsixel')


   x = np.linspace(0, 1)
   y = x**2
   plt.plot(x, y)
   plt.show()
